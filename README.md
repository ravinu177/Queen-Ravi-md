## About

**QUEEN RIAM** is a modern WhatsApp multi-device bot built with **Node.js**, **Baileys**, and **Express**. Designed for simplicity, extensibility, and reliable automation for both group and personal chats.

> Please use responsibly and for educational purposes only.

## Features

| Feature | Description |
|---------|-------------|
| **Command Handler** | Modular command system with support for custom plugins and events |
| **Multi-Device** | Full multi-device support via Baileys |
| **One-Click Deploy** | Deploy instantly on Heroku, Railway, Render, Koyeb, or KataBump |
| **Session Pairing** | Secure, simple session ID generation |
| **Customizable** | Easily configurable via environment variables |
| **Clean Logging** | User-friendly logs and clear error messages |

## Quick Start

```bash
git clone https://github.com/Dev-Kango/Queen-Riam.git
cd Queen-Riam
npm install
npm start
```
